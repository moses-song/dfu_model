## 1) MVP1 범위와 KPI 확정 + 상세 일정

### MVP1 목표
- 모바일 촬영 DFU 이미지 기반 상처 분류 기능의 안정적 제공
- FastAPI /classify를 통한 추론 파이프라인 안정화
- 임상 분류체계(SINBAD/Wagner) 중 1종을 MVP1의 공식 분류 기준으로 채택

### MVP1 범위
- 포함
  - 이미지 업로드 및 기본 검증
  - 분류 모델 추론 (DINOv3 + Mask2Former 기반)
  - 분류 결과 응답 스키마(ClassificationResult)
  - 로그/모니터링 최소셋
- 제외
  - 상처 부위 세그멘테이션 결과 제공
  - 3D 재구성 및 깊이/부피 지표
  - EMR 멀티모달 위험도 산출
  - RAG 기반 설명

### MVP1 KPI (초안)
- 분류 정확도(Overall Accuracy): 내부 검증셋 기준 목표치 설정
- 클래스별 재현율(Recall): 임상적으로 중요한 등급에서 최소 기준 유지
- 응답 지연 시간(P95): 모바일 업로드 기준 응답 SLA 설정
- 오류율: 실패 응답 비율 목표치 설정
- 재현성: 동일 이미지 재요청 시 결과 일관성 확보

### 상세 일정 (초안, 4주 기준)
- Week 1
  - 데이터 정제 및 라벨 기준 확정
  - 모델/서빙 환경 정리
  - API 입출력 스키마 확정
- Week 2
  - 모델 튜닝 및 1차 성능 검증
  - 예외 처리/유효성 검증 강화
- Week 3
  - 통합 테스트 및 성능 테스트
  - QA 시나리오 작성 및 실행
- Week 4
  - 베타 배포 준비
  - KPI 리포트 작성 및 개선 포인트 정리

## 2) 특허 A/B의 claim 범위 초안

### 특허 A: 2D+3D 상처 기하학 기반 시계열 악화 예측 방법
- Claim 1
  - 2D 이미지를 입력받아 상처 영역을 세그멘테이션하고, 동일 대상의 3D 재구성을 통해 깊이/부피/면적 등 기하학적 특징을 추출하는 방법.
- Claim 2
  - 상기 기하학적 특징과 EMR 시계열 데이터(HbA1c, Glucose, BMI 등 3시점)를 결합하여 악화/절단 위험도를 예측하는 방법.
- Claim 3
  - 예측 결과를 임상 지표(절단 확률, 악화 확률 등)로 변환하여 제공하는 방법.
- Claim 4
  - 상기 예측 모델을 모바일 촬영 환경의 데이터 품질 변동을 고려해 보정하는 방법.

### 특허 B: 촬영 품질 보정 + 멀티모달 위험도 예측을 위한 자동 캘리브레이션
- Claim 1
  - 모바일 촬영 이미지의 조명/각도/거리 품질을 자동 평가하고, 기준에 맞게 보정하는 방법.
- Claim 2
  - 상기 보정된 2D 이미지 및 3D 재구성 데이터를 정규화하여 비교 가능한 상처 특징값을 산출하는 방법.
- Claim 3
  - 정규화된 특징값과 EMR 시계열을 결합하여 위험도를 예측하는 방법.
- Claim 4
  - 촬영 품질이 낮을 경우 재촬영을 유도하거나 예측 신뢰도를 조정하는 방법.

### 특허 C: 2D 입력 기반 3D 디지털 트윈 생성 방법
- Claim 1
  - 2D 상처 이미지를 입력받아 상처 부위를 분리하고, 생성형 모델/기하학 모델을 통해 3D 형상을 복원하는 방법.
- Claim 2
  - 복원된 3D 형상을 표준 스케일로 정규화하고, 시점 간 변화량(깊이/부피/면적)을 추적하는 방법.
- Claim 3
  - 상기 3D 디지털 트윈을 임상 의사결정 또는 위험도 예측의 입력으로 사용하는 방법.

### 특허 D: 2D/3D 기반 걸음걸이(보행) 분석 방법
- Claim 1
  - 2D/3D 발 이미지 또는 연속 프레임을 입력받아 발 하중 분포 및 보행 특성을 추정하는 방법.
- Claim 2
  - 상처 위치/크기/깊이와 보행 패턴을 결합해 재발/악화 위험도를 추정하는 방법.
- Claim 3
  - 보행 분석 결과를 환자용 피드백(재활/보행 교정)으로 제공하는 방법.

### 특허 E: EMR 연동 기반 개인화 위험도 산출 및 추적
- Claim 1
  - EMR에서 HbA1c, Glucose, BMI 등 시계열 데이터를 자동 연동·정규화하는 방법.
- Claim 2
  - 상기 EMR 정보와 이미지 기반 지표를 결합하여 개인화 위험도를 산출하는 방법.
- Claim 3
  - 위험도 변화 추이를 의료진 대시보드 또는 환자 앱에 제공하는 방법.
## 3) 특허 아이디어 설명
1. **특허 A: “2D+3D 상처 기하학 기반 시계열 악화 예측 방법”**
    - 핵심 아이디어: 2D 이미지에서 상처 영역을 세그멘테이션하고, 3D 재구성으로 상처 깊이·부피·면적의 기하학적 변화를 산출한 뒤, EMR 3시점의 HbA1c/Glucose/BMI 등을 결합해 악화/절단 위험을 시계열로 예측
    - 차별점:
        - 2D와 3D의 결합으로 “상처 깊이 변화”를 정량화
        - 3시점 EMR 데이터와 결합된 “경향성 기반 예측”
    - 예상 Claim 포인트:
        - 상처의 3차원 기하 특성 추출
        - 기하 특성과 EMR 시계열을 결합한 위험도 예측 알고리즘
        - 결과의 임상 지표(절단 가능성, 악화 확률) 산출
    - 데이터 필요성: 2D+3D 이미지 쌍, EMR 3시점
    - MVP 기여: MVP2~3 기능의 특허화
2. **특허 B: “모바일 촬영 품질 보정 + 상처 위험도 예측을 위한 멀티모달 자동 캘리브레이션 방법”**
    - 핵심 아이디어: 환자 촬영 이미지의 품질(조명, 각도, 거리)을 자동 판정하고, 3D 재구성으로 스케일/각도 보정 후 정규화된 상처 데이터를 생성하여, EMR 시계열과 결합해 위험도를 추정
    - 차별점:
        - 환자 촬영의 ‘품질 보정’ 자체를 핵심 기술로 포함
        - 촬영 품질과 임상 위험도 예측을 하나의 파이프라인으로 연결
    - 예상 Claim 포인트:
        - 촬영 품질 자동 평가 및 보정 알고리즘
        - 보정된 이미지/3D 정보를 기반으로 한 위험도 예측
        - 환자 촬영 환경 변화에도 안정적인 예측 제공
    - 데이터 필요성: 촬영 품질이 다양한 2D/3D 데이터 + EMR 시계열
    - MVP 기여: MVP1~2에서 품질 게이트 및 전처리로 반영 가능
3. **특허 C: “2D 입력 기반 3D 디지털 트윈 생성 방법”**
    - 핵심 아이디어: 2D 상처 이미지 입력만으로 상처 영역을 분리하고, 생성형/기하학적 복원 모델로 3D 형상을 추정해 디지털 트윈을 구성한다. 시점 간 3D 변화량을 추적해 치료 반응을 정량화한다.
    - 차별점:
        - 2D 단일 입력으로 3D 디지털 트윈을 생성하는 경량화 파이프라인
        - 상처 깊이/부피/면적 변화의 시계열 추적을 디지털 트윈으로 통합
    - 예상 Claim 포인트:
        - 2D 기반 3D 상처 형상 복원 방법
        - 디지털 트윈 표준 스케일 정규화 및 변화량 계산
        - 디지털 트윈 기반 치료 반응/예후 지표 산출
    - 데이터 필요성: 2D 이미지 + 일부 3D GT(학습/검증용), 시점별 비교 데이터
    - MVP 기여: MVP2~3의 3D 기능 확장 및 임상 리포트화
4. **특허 D: “2D/3D 기반 걸음걸이(보행) 분석 방법”**
    - 핵심 아이디어: 2D/3D 발 이미지(또는 연속 프레임)를 입력으로 보행 특성(하중 분포, 접지 패턴, 좌우 비대칭)을 추정하고, 상처 위치/크기/깊이와 결합해 악화 위험을 평가한다.
    - 차별점:
        - 상처 정보와 보행 정보를 결합한 재발/악화 위험 모델
        - 의료진 피드백과 환자 피드백을 동시에 제공
    - 예상 Claim 포인트:
        - 2D/3D 기반 보행 특성 추정 알고리즘
        - 보행-상처 통합 위험도 산출 방법
        - 보행 교정/재활 피드백 생성 방법
    - 데이터 필요성: 2D/3D 이미지 + 보행 시퀀스/동작 데이터
    - MVP 기여: MVP3 이후 확장 기능(원격 재활/관리 연동)
5. **특허 E: “EMR 연동 기반 개인화 위험도 산출 및 추적”**
    - 핵심 아이디어: EMR에서 HbA1c/Glucose/BMI 등 시계열 데이터를 자동 연동·정규화하고, 이미지 기반 상처 지표와 결합해 개인화 위험도를 산출·추적한다.
    - 차별점:
        - 자동 EMR 연동과 위험도 추적의 end-to-end 파이프라인
        - 개인화 위험도 변화 추이를 의사결정에 직접 활용 가능
    - 예상 Claim 포인트:
        - EMR 시계열 자동 수집/정규화 방법
        - EMR+이미지 융합 기반 위험도 모델
        - 위험도 변화 추이 시각화/알림 방법
    - 데이터 필요성: EMR 시계열 + 이미지 지표 + 사용자/의료진 피드백 로그
    - MVP 기여: MVP3의 멀티모달 위험도/대시보드 기능과 직결
